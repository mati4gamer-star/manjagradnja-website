## O projektu

Prezentacijski sajt za **Manju Gradnju** — građevinsku firmu specijalizovanu za novogradnju, renoviranje i adaptaciju stambenih i poslovnih objekata u Srbiji.

Ovo je **prvi klijentski projekat** agencije [HoriZEN](https://github.com/mati4gamer-star).

## Tehnički stack

| | |
|---|---|
| **Frontend** | Vanilla HTML5, CSS3, JavaScript |
| **Slike** | WebP + JPEG fallback, lazy loading |
| **Animacije** | CSS transitions, Intersection Observer |
| **Deploy** | Vercel + custom domen |
| **SEO** | Schema.org, Open Graph, sitemap.xml |
